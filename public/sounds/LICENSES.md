# 键声音效来源（均为 CC0）

## 1. Cherry KC 1000 单键实录 — `mx-brown/`、`mx-blue/`

- **作者：** unicaegames（OpenGameArt）
- **包名：** Keyboard Soundpack #1
- **许可：** [CC0](https://creativecommons.org/publicdomain/zero/1.0/)
- **来源：** https://opengameart.org/content/keyboard-soundpack-1-typing-and-single-keystrokes  
- **说明：** 同一套 32 个单键采样；`mx-brown` 使用 keypress 001–004 与 005（退格），`mx-blue` 使用 017–020 与 021（退格），以区分听感。

## 2. Kenney Interface Sounds — `soft/`

- **作者：** Kenney (kenney.nl)
- **许可：** CC0 — https://creativecommons.org/publicdomain/zero/1.0/
- **来源：** https://opengameart.org/content/interface-sounds  
- **说明：** 自 OGG 经 `ffmpeg` 转为 44.1kHz 单声道 WAV；使用 `tick_001`–`tick_004`（无 `tick_003`）、`select_001`、`back_001` 等。

## 3. 输错字母 — `error.wav`

- **来源：** 程序合成（`node scripts/generate-key-sounds.mjs --error-only`），与机械键声无关；可与实录键声并存。
